VR Games

Since the beginning of 2021, I started experimenting with VR Game development

  • I created several games using Unreal Engine 4 and Unity
  • Most of my games support OpenXR which means that they can be used in Steam VR, HTC Vive, Oculus Quest 2, etc.
  • Use Handtracking input for my Oculus Quest 2 game prototypes
  • Adapt standard game functionalities such as locomotion, storage system and triggers to VR
  • Work in group to design a VR game and make changes.
  • Designed a virtual simulation for students to learn how a manufacturing robot works
